# Alya Client

## Introduction

Alya Client is a free, open-source Minecraft hacked client for 1.8.9.

Alya is designed to be easy and familiar to use while providing powerful features like a scripting API and running on toasters.

Alya's UI is devoid of special effects and focues on high-performance and compatability, the most fancy thing is the main menu background. Once in game the UI is beautifully simple and stays out of your way.

## Website

You can visit the website [here](https://alya.thoq.dev)

## Notable Features
- Powerful scripting API
- Optimized
- Converted to LWJGL3
- Familiar feel
- Stays out of your way
- Open-source, free software under the `GPL-2.0-only` license
- First-class Linux support
